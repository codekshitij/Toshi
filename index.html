RAG architecture

Research-grade retrieval,
not keyword search.

01
Ingestion
Downloads the actual 10-K HTML from EDGAR, strips iXBRL markup, and extracts named sections: Business, Risk Factors, MD&A, Financials.
02
Chunking
400-word overlapping chunks with 50-word overlap. Each chunk stores its full parent section for context expansion.
03
GPU Embeddings
all-MiniLM-L6-v2 runs locally via sentence-transformers with automatic MPS / CUDA / CPU detection. 384-dimensional vectors.
04
MMR Retrieval
Maximal Marginal Relevance balances relevance and diversity. Avoids returning five near-identical chunks.
05
CRAG Self-Critique
Corrective RAG scores each chunk against the query. Low-confidence chunks are discarded or sentence-extracted automatically.
06
Cross-Encoder Reranking
ms-marco-MiniLM-L-6-v2 scores every (query, chunk) pair jointly for a final precision-ranked top 5.

Get started

Up and running
in 5 minutes

01

Clone and install

Python 3.11 required. All dependencies install automatically.

git clone https://github.com/yourusername/toshi.git
cd toshi
python3.11 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
02

Set your email

SEC EDGAR requires a contact email in the User-Agent header. That's their only requirement.

cp .env.example .env
# Edit .env and set:
SEC_USER_AGENT="Toshi your@email.com"
03

Connect Claude Desktop

Edit claude_desktop_config.json and point it at TÅshi.

{
  "mcpServers": {
    "toshi": {
      "command": "/path/to/toshi/venv/bin/python3",
      "args": ["/path/to/toshi/server.py"]
    }
  }
}
